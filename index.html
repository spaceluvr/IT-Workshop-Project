What Is This

This is the first of many webpages of the gaming website made as part of the ICS 111: IT Workshop course. The website contains:

  • Home Page
  • Game 1: Find The Hidden Object
  • Game 2: Rock, Paper, Scissors
  • Game 3: Quizzzz
  • About
  • Rules

Home Page

The Home Page consists of a navigation bar/header at the top where the name of the website is displayed. It also contains the About and Rules links, clicking which the user is redirected to the respective page.
The bottom part of the webpage contains the games. Hovering over the buttons shows a brief description of the game as well as makes the button appear larger and the changes the cursor to a pointer.

Game 1: Find The Hidden Object

The first game is a simple game that tests how accurately a peson can notice the hidden details in an image. it consists of 5 images out of which one is randomly displayed to the player. They are then asked to find a particular object in the image.
If they are able to do so, an alert box informs them that they are correct and then redirects them to a page which asks them if they want to play again. If they click on yes, the cycle repeats again and clicking on no takes them back to the Home Page

Game 2: Rock, Paper, Scissors

The second game is a classic rock, paper, scissors game. The player selects their offence and the computer then chooses a random offence, displaying both on screen.
Based on what was chosen, an alert box informs the player whether they won, lost or if it was a tie. It then asks if they want to play again. If they click on yes, the cycle repeats again and clicking on no takes them back to the Home Page

Game 3: Quizzzz

The third game is a straightforward quiz game consisting of 6 questions, each with 4 possible answers, displayed randomly to the player.
On clicking the correct answer, a webpage with a clapping audio informs the player that their answer was correct. On clicking the wrong answer a webpage informs them that their answer was wrong.
These pages also ask the user if they wish to play again. If they click on yes, the cycle repeats again and clicking on no takes them back to the Home Page

About

The About tab on the Home Page changes color and the cursor turns to pointer when the mouse is hovered over it.
The page tells basic information about the website such as what it is, how it's made etc.
The About tab in the navigation bar is replaced with the Home tab for easy access to the Home Page

Rules

The Rules tab on the Home Page changes color and the cursor turns to pointer when the mouse is hovered over it.
The page informs the players of the basic rules of the games present on the website.
The Rules tab in the navigation bar is replaced with the Home tab for easy access to the Home Page
