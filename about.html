What Is This

This is a gaming website initially containing 3 games. The existence of this website can be owed to the first semester project assigned in the ICS 111: IT Workshop course at IIIT Kottayam

How Is It Made

The website is made using HTML, CSS, and JavaScript elements.
HTML provides the basic structure
CSS allows for designing and beautification
JavaScript helps in the functioning of the games

Learning

The games themselves while look simple on the outside, employ lengthy JavaScript code and heavy CSS styling. Whether that be marking pixels, figuring out the logic, or making sure the alignment is correct, making the website taught a lot about web development and how much effort actually goes into making large websites
